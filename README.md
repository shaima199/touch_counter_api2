# Touch Counter API 2

This project is part of an AI-based football player analysis system.

It processes video input to analyze player performance using computer vision and machine learning techniques.

Main features:
- Count ball touches
- Measure player stability
- Calculate skill score
- Classify player level

Technologies used:
- Python
- FastAPI
- OpenCV
- MediaPipe
- YOLO
- NumPy
- Pandas
